What's New in Nick Spoofer 1.06

WHAT'S NEW IN 1.06
  • Now supports Brood War 1.08 final.
  • Choose Version dialog removed.
  • Colors box now includes the colors that it will be in both the pre-game and in-game sessions.
  • Now minimizes to the system tray for memory conservation.
  • Supports Windows NT/2000/XP. Previous versions of the Nick Spoofer would not work on the NT architecture due to process permissions (you would get an Error 02). This has been fixed in 1.06.
  • Tooltip help is now included. Hold your mouse over any feature of the Nick Spoofer to get a description of its uses.
  • Bug fix: No more "Overflow" errors on program launch. This was due to screen coordinates beyond the integer maximum.
  • Bug fix: Nick Spoofer no longer launches off-screen, rendering it invisible.
  • Bug fix: Nick Spoofer no longer changes your name to "Error 01" when launching the Nick Spoofer before you launch Brood War.

WHAT'S NEW IN 1.05a
  • Complete rewrite since I lost the original source. Notify me of any regressions.
  • Redesigned the interface and merged configuration into the Advanced tab rather than a separate application.
  • Nick Spoofer now saves your last settings as to whether or not advanced mode is expanded and the position on the screen.
  • Now compatible on 1.07 and 1.08 Revision O! Full support for 1.08 final will be available in Nick Spoofer version 1.05b.
  • Removed "Switch to Brood War After Change" since it never really worked anyway.
  • Compiled HTML Help file now included.

WHAT'S NEW IN 1.04
  • Two new options added to configuration:
    • "Retain Nickname Throughout Session" - Normally, Starcraft resets your spoofed nickname after you leave a game. This keeps your current name until you change it.
    • "Disable Nickname Checking Interval" - Recommended only if you have problems with the Nick Spoofer checking your nickname every 2 seconds. Click the Retreive box to manually check your nickname with this option enabled.
  • Fixed "Switch To Brood War After Change" option - it now completely restores Brood War, however, it does so by hiding the window and restoring it (due to issues otherwise). You may not wish for this to happen, so disable this option if it causes problems.
  • Found that spaces may trigger Battle.net bans. Spaces may not be inserted into a name without the "Use Colors" box being checked.

WHAT'S NEW IN 1.03
  • Added a Configuration tool with the following options:
    • "Clear Nick Box On Change" - clears the textbox containing the new name after submission.
    • "Switch To Brood War After Change" - Switches to the Brood War window after submission, however, it currently doesn't restore the window.
    • "Always Show Color Warning" - To always remind you to read the readme.txt file, for those who may have others accessing the file.
    • Enable "Safe Mode" - Disable colors, previously activated by downloading a .reg file.
  • Fixed color insertion bug to insert color after the caret instead at the end of the box.

WHAT'S NEW IN 1.02
  • Required text in the Modify box to prevent problems from within Starcraft.
  • Added a "safe" mode. Download the .reg file from www.scbackstab.com to activate the no-color mode (useful if you're on a static IP and don't want to be banned).
  • Increased maximum length to 21 characters, from 20.
  • Removed the manual updating of your battle.net name in favor of the name automatically updating every 2 seconds.
  • Use of registry to display the color warning box, will only show once (not once per session).
  • Changed frame labels to be more descriptive ("Retreive" and "Modify").
  • Changed program icon.
  • Changed name to Nick Spoofer to match the more commonly-used name.

WHAT'S NEW IN 1.01
  • Fixed space bug in earlier versions where your name would have a space before the ":" colon.
  • Fixed earlier bug which the current Battle.net name would only display the first 15 characters.
