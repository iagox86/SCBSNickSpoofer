Troubleshooting

Error Codes
  • Error 01: Brood War window not found. Brood War must be open to spoof your nickname. - Open Brood War, connect to Battle.Net and try again.
  • Error 02: Unable to retreive Brood War process handle. - Reboot Windows and try again. If the error persists, the Nick Spoofer may not be compatible with your system.
  • Error 03: You must connect to Battle.net. - You cannot read your nickname without first connecting to Battle.Net.
  • Error 04: No data entered into Brood War. You must be connected to Battle.net to spoof your nickname. - You cannot spoof your nickname without first connecting to Battle.Net.
  • A required DLL file, MSVBVM60.DLL, was not found. - You must first download the Microsoft Visual Basic 6.0 Runtime DLL files before you can use the Nick Spoofer. Available here.
  • File mscomctl.ocx not found: file is missing or invalid. - You must download the Microsoft Windows Common Controls 6.0 before you can use the Nick Spoofer. Download mscomctl.ocx and place it in your system directory. The file is available here.
